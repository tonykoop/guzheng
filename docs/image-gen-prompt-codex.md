# Image Generation Prompt

Concept visualization only; not fabrication authority.

## Hero Prompt

Photorealistic 3/4 studio product photograph of a guzheng reference-member prototype, a long plucked board zither with a gently crowned soundboard, moveable bridges in a clear bridge corridor, end blocks, many parallel strings, and a restrained natural-wood finish. Show the instrument as the single reference build rather than a full family: clean bridge templates, visible string paths, smooth side rails, and a workmanlike setup suitable for measuring crown, creep, bridge travel, and load. Neutral studio background, soft directional lighting, realistic table-scale cues, about a 50mm lens, high detail, photorealistic.

## In-Context Prompt

Photorealistic workshop scene of the guzheng reference member on a long bench, with a player lightly plucking near the bridges while a builder observes bridge corridor clearance and soundboard crown. Natural light, bridge templates and a small deflection gauge nearby, no performance-stage polish, focus on validation and playable reference evidence.

## Detail / Macro Prompt

Macro photorealistic close-up of several moveable guzheng bridges under parallel strings on a crowned wooden soundboard, showing bridge feet, bridge spacing corridor, string contact points, subtle wood grain, and a tiny shadow gap under each bridge foot.

## Negative / Avoid

No text, no watermarks, no fabricated dimensions, no impossible bridge overlap, no decorative claims from concept art, no extra hands or unrealistic finger positions.
